1 Write a C program to read “n” value and display following pattern

Test Case: 1
Input : 
Enter n value:6
Output : 
 6
 6 5
 6 5 4
 6 5 4 3
 6 5 4 3 2
 6 5 4 3 2 1


Program:
#include<stdio.h>
 int main()
 {
     int i, j, rows;
     printf("enter the number of rows:");
     scanf("%d", &rows);
     for(i=1; i<=rows; ++i)
     {
         for(j=1; j<=i;++j)
         {
             printf("%d", j);
         }
         printf("\n");
     }
     return 0;
 }
